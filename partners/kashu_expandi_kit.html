Expandi Campaign Setup
One campaign per persona. Run 2-3 simultaneously. Follow this exact sequence to stay safe.
Priority order: ISO → CPA → Bookkeeper first. These have the highest conversion to active MRPs. Stand up 3 campaigns in parallel on day one, then add the rest week 2.
New Expandi account: start at 10 requests/day for the first 2 weeks before going to 20-25. LinkedIn flags sudden spikes. If your account is warmed up already, go straight to 20.

Campaign Build Steps

1
Create a new Campaign
Expandi dashboard → Campaigns → New Campaign → Connection Campaign. Name it: MRP — [Persona] — [Date] e.g. "MRP — CPA — Mar 2026"
2
Set the Lead Source
Select "LinkedIn Search" → paste in the search URL from the persona tab. Expandi will scrape the list automatically. Aim for 300-500 profiles per campaign.
3
Configure the sequence
Add 3 steps: (1) Connection Request with note → (2) Wait 3 days → (3) Message on accept → (4) Wait 5 days → (5) Follow-up message if no reply. Copy from each persona tab.
4
Set daily limits
Campaign settings → Max invitations per day: 20 → Enable random delays → Working hours: 8am–6pm your timezone → Weekdays only.
5
Enable Smart Limits
Expandi Settings → Smart Limits → ON. This auto-pauses if LinkedIn shows friction. Never turn this off.
6
Launch and monitor day 1
Hit Launch. Check back after 2 hours. If you see a LinkedIn "unusual activity" warning, pause immediately and let it rest 48 hours. Normal behavior: smooth sends, no warnings.

Setup (one-time, ~2 hours)

1
Get a secondary domain
Buy kashupayments.com or kashupay.co or getkashu.com. Never cold email from kashupay.com — it will hurt deliverability on your primary domain. $12/year on Namecheap.
2
Set up Instantly.ai
Connect your secondary domain. Add 3-5 sending inboxes (john@kashupayments.com, j.polimeni@kashupayments.com, etc). Enable email warmup — run for 2 weeks before sending. Limit: 30-50 emails/day per inbox to start, scale to 100 after 30 days.
3
Build lead lists in Apollo.io
Filter by: Job Title (CPA, Bookkeeper, Payroll Manager, Insurance Agent, etc.) + Country: US + Company size: 1-50 employees. Export 500-1,000 contacts per persona. Verify emails before importing.
4
Load sequences into Instantly
One campaign per persona. Paste subject line + email body. Set sending window: 8am-5pm, Mon-Fri. Enable open/reply tracking. Connect reply → GHL workflow via Zapier for instant follow-up.
